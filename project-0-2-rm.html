Type

Team Project

Work Period

Oct 2019 (2 months | 8 hours a day)

Description

Relationship Managers role is to go to the end user and collect data and documents as necessary and submit them to the bank or distributor. But there are a lot of problems that are currently unresolved in this process. Some of them include revisiting the user for some documents that may have missed or not clear, spelling mistakes in the application and improper address details.

Using the RM Assisted Mobile app, productivity of evry individual RM can be increased and tracked. With digital submission of documents to geo location and live verification, redo of applications and rejection rate can be largely minimised.
