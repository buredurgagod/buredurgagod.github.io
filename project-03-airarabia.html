iOS Application of Air Arabia

UI/UX Material Design Graphic Design iOS iOS Project

My role

UI/UX Designer | Frontend Developer

Technical Stack

HTML, CSS, JS, jQuery, Materialize CSS, CKEditor

Type

Team Project

Work Period

May 2016 (2 months | 4 hours a day)

Description

Air Arabia is low cost airlines whose main base is Sharjah International Airport. It already has an app where the UI and UX is not so satisfactory. The present app also lacks many vital features. So the aim is to make a new design which has all the missing ones. Taking all these points into consideration the design was made. The UI has totally been changed and also many features were included. Some of them are fare alert, disability assistance, e-ticket, etc. The color scheme throughout the app remains the same since it is their color theme.
