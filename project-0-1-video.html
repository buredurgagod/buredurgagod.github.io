Type

May 2017 (2 months | 8 hours a day)

Description

As per new RBI guidelines, it is mandatory for a live video verification of the end user in case of digitally onboarding the user. This video will be recorded and saved for future use in case of legal problems. So, there is a need for video verification portal.

As soon as the user completes the application, he will be asked to schedule video verification slot or wait for a few minutes to connect to the next available slot. Once the video verification is done, this video data will be saved and sent along with the application to the backops user for verification. The backops user takes a right decision based on all the available data and the video verification data.
