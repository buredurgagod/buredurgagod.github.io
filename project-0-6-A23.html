Description

Wizely aims to guide financially new users on to a path of financial wellness. This is achieved by giving tips to users on how to spend, save and do other activities. Users keeping a good spending and saving behaviour are allocated Wellness score. Wellness score is a measure of how financially well the user is and helps Wizely give recommendations to their users.

Using recommendations provided by Wizely, users can improve their financial wellness score and a good behaviour is also rewarded by scratch cards.
